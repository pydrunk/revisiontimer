Revision Timer is a simple application that (believe it or not) is a timer, the timer can be in 3 states, chilling, revising and paused. When the timer is chilling the timer goes down, when the timer is revising the timer goes up, when the timer is paused nothing happens.

This is meant to be used to balance revision and chilling, this is why there are also options for how fast the timer increases when your revising and decreases when your chilling (e.g if you want 1 hour of revision every 2 hours of chilling, you would make the revision increase
2 and the chilling decrease 1)

also when the timer goes to 0 a noise plays.
